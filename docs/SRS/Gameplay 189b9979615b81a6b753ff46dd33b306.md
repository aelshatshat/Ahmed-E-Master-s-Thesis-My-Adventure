# Core Gameplay

# Controls

WASD for movement

Left Click for basic attack

Right Click for spell

Shift for a dodge ability

Spacebar for jump

- Double jump

Jump → Attack does an Airborne Sword Spin

# Objectives

## Primary Game Loop

- Move
- Attack
    - Four hit combo
- Cast spell
    
    Spells:
    
    - Instant Cast
        - Fireball
    - Incantation (cast time)
        - Magic Missilerack
    - Combo Attack (attacks match combo attack stages)
        - Magic Explosion
- Jump
- Jump Attack
- Dodge

## Secondary Game Loop

- Avoid boss abilities
- Damage boss with your abilities

## Long-Team Goals

- Kill bosses
- Beat game
